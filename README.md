# XJTU-Verilog-Moniter
XJTU微电子专业数字系统设计基础配套实验大作业项目（废稿未用于答辩，但是完成度颇高）
## 1. 背景介绍
本项目是西安交通大学微电子专业在3-2学期开设的专业选修课“数字系统设计基础”的配套实验的最后一个实验项目——基于 FPGA 开发平台的自主综合设计。这一部分要求学生根据FPGA平台的硬件资源和自我兴趣自行设计实验内容、实验方案，完成实验设计、代码实现、仿真验证、功能演示、报告撰写。本项目是我为此编写的一个比较简单烂怂的兜底项目，由于最后一天组里的大爹成功地搓出了高大上的深度学习项目代码，因此本项目被废弃，没有参与答辩。既然如此，学弟学妹可以……学习一下，杜绝学术不端行为喵！
## 2.模块功能
本项目可在VGA屏幕上显示CPU和内存相关信息，并配备实时进度条展示利用率，屏幕的四角可以展示个人logo，可使用两个按钮调节屏幕亮度（功能不完善）。各模块功能如下：
1. top_stage2：顶层模块
2. uart_rx：获取来自PC端的CPU和RAM数据
3. data_parser：将数据输出到VGA或指示灯
4. vga_timing：为vga扫描提供时序信号
5. vga_render：在屏幕上绘制ui
6. mini_font_rom：vga_render的子模块，存储字体信息
7. seg_display：在数码管上显示数字，用于调试，可酌情删除
8. top_stage1：项目的早期调试用顶层模块，需禁用。

除Verilog模块外，本项目还需要使用到"monitor.py"和"make_logo.py"两个程序，分别用于获取并发送硬件信息和生成64*64规格的图片（可选择是否需要去背景）（显然你需要python环境）。
## 3. 使用说明
### 3.1 准备工作
将"make_logo.py"和你需要显示在四角的logo图片置于同一个文件夹，修改"make_logo.py"中的图片位置，生成txt文件，将生成的txt文件移动到项目文件夹中使得Verilog代码可以访问。
### 3.2 上电测试
连接屏幕、开发板和PC，开发板上电后，烧录bit流文件，运行"monitor.py"即可。
